# JSON Commands

SpinelDB supports storing and manipulating JSON documents.

| Command | Usage | Summary |
| --- | --- | --- |
| `JSON.DEL` | `JSON.DEL key [path]` | Deletes a value from a JSON document. |
| `JSON.GET` | `JSON.GET key [path]` | Gets a value from a JSON document. |
| `JSON.SET` | `JSON.SET key path value` | Sets a value in a JSON document. |
| `JSON.TYPE` | `JSON.TYPE key [path]` | Reports the type of JSON value at path. |
